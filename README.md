## Instalação 
**Linguagens**: Haskell<br>
**Tecnologias**: GHCI<br>

- Instalar GHC para seu computador.
- Rodar o seguinte comando dentro da pasta `src`.

```
ghc --make -o menu menu.hs -o blackjack blackjack.hs
```

- Executar o binario `blackjack` gerado.
